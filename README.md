# Responsive Landing Page

This project is a modern, fully responsive landing page developed as **Task-01** of the **Prodigy InfoTech Web Development Internship**.

## Overview

The landing page features an interactive navigation menu that remains fixed while scrolling and dynamically changes its appearance based on the user's scroll position. The design is responsive, ensuring an optimal experience across desktop, tablet, and mobile devices.

## Features

- Responsive layout
- Fixed navigation bar
- Scroll-based navbar styling
- Interactive hover effects
- Smooth scrolling navigation
- Modern and clean user interface
- Cross-browser compatibility

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
